PydentityCore documentation
===========================

Installation
------------
You can install ``pydenticore`` with ``pip``:

.. code-block:: console

    $ pip install pydenticore

You can additionally install ready-made models and storage.

For SQLAlchemy:

.. code-block:: console

    pip install pydentity-db-sqlalchemy

or

.. code-block:: console

    pip install pydenticore[sqlalchemy]

For TotroiseORM:

.. code-block:: console

    pip install pydentity-db-tortoise

or

.. code-block:: console

    pip install pydenticore[tortoise]
